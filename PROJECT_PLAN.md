# PROJECT PLAN - Sirius

## 2. MVP (Minimal Viable Product)

Il prodotto minimo funzionante includerà:

- Sistema di login utente
- Visualizzazione dati da database
- Inserimento dati tramite form
- Comunicazione tra frontend e backend (API REST)

Obiettivo:
Realizzare una prima versione semplice ma completa che dimostri il funzionamento dell'intero sistema.

---

## 3. Work Breakdown Structure (WBS)

### Setup iniziale
1. Creazione repository
2. Clonazione progetto
3. Setup ambiente di sviluppo

### Backend
4. Creazione server
5. Connessione al database
6. Creazione API REST
7. Test API con Postman

### Frontend
8. Creazione interfaccia base
9. Collegamento API
10. Gestione form
11. Visualizzazione dati

### Integrazione
12. Collegamento frontend-backend
13. Test completo
14. Debug

### Finalizzazione
15. Miglioramenti UI
16. Documentazione
17. Eventuale deploy
